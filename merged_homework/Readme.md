# Cypress with docker (HW 3.2 && 3.3)

Docker compose is used to run concurrent tests in three different browsers (firefox, electron, chrome)

## How to run

```bash
npm install
```

Then run

```bash
docker-compose up
```

You can see recorded test videos under *cypress/videos*
